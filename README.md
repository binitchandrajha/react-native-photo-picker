# react-native-modern-photo-picker

A Kotlin-based React Native native module that provides access to Android's modern Photo Picker API.

## Installation


```sh
npm install react-native-modern-photo-picker
```


## Usage


```js
import { multiply } from 'react-native-modern-photo-picker';

// ...

const result = multiply(3, 7);
```
